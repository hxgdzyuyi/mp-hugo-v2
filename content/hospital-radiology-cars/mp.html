01战火催生的医疗革命

一战使大规模伤亡成为残酷现实，也迫使医疗技术、设备配置与救治流程迅速革新。传统的术前判断常依赖伤口位置、体表标记、触诊和外科经验推断；但在炮弹造成的复杂骨折、弹片残留、深部损伤和感染风险面前，这些方法往往不足以准确判断伤情。

在这种背景下，X 射线凭借非侵入性和直观成像的优势，成为战时外科诊疗的重要工具。它能够帮助医生确认子弹、弹片是否仍在体内，判断骨折形态、碎骨分布、关节状态和愈合进程，也能在必要时为手术提供精准的定位依据。

居里夫人参与并推动了法国战时放射服务的组织工作。她协助建立医院放射室、配置放射科医疗车、培训操作人员，并把原本集中在少数大城市的放射技术普及到前线医院和后方救治机构。科学不再只是实验室中的发现，而成为战场上保护生命、减轻痛苦的医疗力量。

02医疗设施和放射科医疗车

战时放射室的搭建

建设战时放射室，首要难题是电力供应。如果医院本身具备 100 至 200 伏的交流电，便可直接接入变压器；若前线缺乏电力，则必须依靠汽油或燃气发电机组。通常，一台 110 伏、25 安培的发电机组就足以支撑普通放射站的运转，许多前线医院正是凭借这类设备，同时解决了基础照明与 X 光检查的用电需求。

此外，设立固定的放射室还需要一间尽量宽敞且能完全遮光的房间。室内需配备变压器、控制台、断续器、X 光检查台、球管支架、荧光屏以及摄影暗室。暗室专用于底片的显影与定影，理想状态下应配备独立水源和暗室专用照明。正如居里夫人在其著作中所述，在熟练人员的操作下，通常只需一到两天，就能搭建起一个功能完备的战时放射室。

什么是放射科医疗车？

放射科医疗车是将完整 X 射线检查设备装载到汽车或卡车上的战时放射站。它通常携带电源装置、变压器、控制设备、X 射线球管、阀管、荧光屏、底片、暗室用品、防护用品和必要工具。抵达医院或救护站后，团队把设备搬入可遮光的房间，连接发电机或车辆带动的发电装置，即可开始检查。

这些车辆不是简单的运输工具，而是可随时补位的移动医疗基础设施。在熟悉地点且过程顺利时，一辆放射科医疗车的设备可在半小时内完成布置；即便条件复杂，通常也很少超过一小时。

放射科医疗车团队的工作

一支典型放射科医疗车团队由医生、操作员和司机组成。车辆接到紧急需求后，带上设备与燃料前往医院；到达后，司机准备发电机或车载发电装置，操作员安装检查台、球管支架、球管、阀管和连接线，医护人员协助遮光并安排伤员。随后开始透视检查、拍片、记录结果，有时还会配合外科医生在 X 光控制下进行操作。

居里夫人在书中多次写到这些团队的主动性、韧性和协作能力。放射科医疗车团队在高压环境中积累了大量经验，也在固定放射站逐渐增多后，转而承担巡回指导、故障排查和人员培训等工作。

放射科医疗车 （VOITURES RADIOLOGIQUES）

战争初期，放射设备极为不足，固定放射室优先设在大型医院；大量临时医院、红十字会辅助医院和前线救护机构既没有电力，也难以迅速建立放射室。放射科医疗车正是在这种情况下成为临时而有效的解决方案。

居里夫人本人曾组织装备 18 辆轻型放射科医疗车，资金来自私人捐赠和“全国伤员援助会”等渠道。与此同时，法国军医部门也陆续装备了配有发电机组的放射科医疗车。战争后期，随着固定站和半固定站增加，放射科医疗车不再总是长途奔赴，而是在许多地区转为驻点服务或应急支援。

放射科医疗车的关键不在华丽车厢，而在“能到达、能供电、能遮光、能成像”。有些车辆用自身发动机带动发电机，减轻了设备重量，使车辆更轻、更快；有些车辆则搭载独立发电机组，更适合长时间停驻工作。车厢还需固定设备，并在必要时形成可临时显影的暗环境。

据原文统计，到 1918 年末，法国军队和本土医院已有 500 多个固定或半固定放射站，约 300 套装在车辆、消毒卡车或汽车外科救护车上的移动设备。1917 至 1918 年间，约 90 万名伤员接受过 X 射线检查，总检查次数约 110 万次。

03X 光是什么？

早期放射学装置的核心并不神秘：先把普通电源升为高压电，再用高压推动 X 射线管工作。管内电子被加速后撞击靶面，释放 X 射线；射线穿过人体，不同密度和组成的组织留下不同阴影，最终在荧光屏或胶片上成像。

低压电 → 变化的一次电流 → 电磁感应 → 高压电 → X 射线管 → 人体成像

1. 先把低压电变成高压电

电源可提供约 110V 的低压电。一次电路中，变阻器 R 调节电流，电流表 A 测量一次电流，开关/保险 C 负责接通与保护，中断器 I 让一次电流周期性断开。

器件链路：电源 110V→ 变阻器 R→ 电流表 A→ 开关 C→ 中断器 I

变化的一次电流进入变压器 T：一次线圈 T1 多用粗线，二次线圈 T2 多用细线，二者由绝缘隔开并通过铁芯耦合。一次侧电流变化后，二次侧便感应出驱动 X 射线管所需的高电压。

2. 高压推动 X 射线管工作

变压器二次侧输出高压后，还需要通过阀管 S 抑制不适合球管工作的反向电流，并由毫安表 MA 测量真正有用的管电流。若反向电流进入 X 射线管，会造成工作不良并损伤球管。

成像链路：高压 → 阀管 S → 毫安表 MA → X 射线管/克鲁克斯管

管内电子在高压下从阴极射出，撞击反阴极靶面后产生 X 射线；射线穿过人体，在荧光屏或胶片上形成骨骼、软组织和金属异物的阴影。

3. “硬度”决定穿透力

早期装置常用火花隙判断等效电压。火花能跨过的距离越长，代表电压越高，球管越“硬”，射线穿透力也越强。

等效火花越长 → 电压越高 → 射线越硬 → 穿透力越强

软 X 射线

电压较低，穿透力较弱，较难穿透人体深部组织。

中等硬度 X 射线

等效火花约 10 厘米，电压约 50,000 伏；适合战时常用的透视与摄影。

硬 X 射线

电压更高、穿透力更强；早期也用于放射治疗。

4. 早期设备的核心原则

1先把低压电变成高压电。

2利用感应、开断与阀管保护，让高压电驱动 X 射线管。

3通过电压高低调节 X 射线硬度与穿透能力。

这些历史设备需要精细调节球管内残余气体，也需要经常维护、清洁接点和检查连接；但它们可以被制成可搬运的战地放射设备，使 X 光检查走出实验室，进入前线医疗现场。

04战时放射医学的技术应用与人员培养

知道 X 光如何产生，只是第一步。真正进入战地医院后，放射医学还要回答另一组问题：怎样快速筛查伤员，怎样避免投影误差，怎样把影像变成外科手术能使用的定位信息，以及谁来操作这些设备。

战地应用链路：伤情判断 → 透视快筛 → 决定摄影 → 正位/侧位或双角定位 → 外科协作 → 防护与培训

1. 为什么战地医院离不开 X 光

在战争医院中，X 光最重要的用途之一，是确认子弹、弹片等异物是否仍在体内，并尽量准确地判断其位置。它也用于观察骨折形态、碎骨、关节状态、骨痂形成和愈合过程，帮助外科医生避免盲目探查。

适用对象：枪伤炮弹伤骨折弹片残留术前定位

2. 成像会放大、变形，关键在投影几何

X 光成像来自近似点状的焦点，因此形成的是锥形投影，而不是完全平行的投影。结果是：影像会放大和变形；当射线倾斜，或目标离胶片、荧光屏较远时，误差会更加明显。

投影链路：焦点→ 锥形 X 光束→ 目标/肢体→ 胶片/荧光屏

要点：距离越远、角度越斜，误差越大；应尽量采用垂直投影，让患部贴近胶片或荧光屏，并用光阑约束照射范围。

3. 透视与摄影：快筛和精查配合

更合理的流程：透视先行 → 初步判断损伤/异物位置 → 决定是否摄影 → 摄影保存与描画

透视速度快，适合战时大批伤员的初步筛查。它可以先找出损伤或异物的大致位置，再决定是否进一步拍片。摄影更适合保存为病历文件，能提供更细致的影像，也能减少操作者长期贴近荧光屏的风险；对细小骨折、脱钙骨和新生骨痂，摄影通常比透视更可靠。

4. 骨折检查与弹片定位

骨折检查

判断骨折严重程度、骨片数量、移位方向及愈合情况。常用标签：正位、侧位。

弹片定位

单张片子不能确定异物真实位置。定位必须结合两个不同视角，或使用移动球管取得双影像，再通过几何方法估算深度和相对位置。

定位还需要解剖判断：弹片与骨骼、肌肉、关节、血管神经之间的关系，往往比几何坐标本身更能决定手术路径。

5. 手术辅助有价值，但不能只靠器械

辅助器件：放射罗盘探针电磁铁手术器械X 光屏幕

放射罗盘、探针、电磁铁等工具都能辅助取弹，但器械不能替代操作者的判断。居里夫人特别提醒，若缺乏预先定位、光阑控制和间歇照射，只凭屏幕上的阴影操作，反而会增加失败、感染和辐射伤害风险。

6. 风险与防护：看不见的伤害

X 光并非无害。它可能导致放射性皮炎，严重时可发展为坏疽甚至死亡；更危险的是，照射时往往没有疼痛，损伤却会延迟出现。

防护标签：铅玻璃屏铅围裙防护眼镜防护手套光阑宽敞房间

核心规则：只在必要时开射线，尽量缩小照射范围；操作者绝不能让裸手的阴影出现在荧光屏上。

7. 最关键的变量：人

1人员短缺与战时现实：操作者要懂诊断，也要懂设备、会调试，并能在现场解决问题。

2医生、操作员与外科医生必须协作：设备读数、投照角度、影像判断和手术计划要相互校正。

3系统化培训补足前线需求：课程训练放射电学、设备管线、透视与摄影方法、解剖和读片，培养能独立工作的人员。

Val-de-Grâce 军医院的放射医生培训约有 300 名医生参加；居里夫人在镭研究所组织的女性操作员学校，则从 1917 年至战争结束培养了约 150 名女性放射操作员，许多人随即进入军队或本土医院服务。

★　战时放射医学的成功，不只来自 X 光设备，更来自技术方法、外科协作、严格防护和系统化人才培养。

05其他

战争证明，X 光不应再是少数地点的例外手段，而应成为医院、救济院、疗养院、乡村救护站和放射科医疗车共同承担的常规诊断能力。它可用于照护退伍伤病者、重建区工伤者、儿童发育检查，以及结核病防治中的肺部病灶和骨性病灶诊断。

放射技术随后也延伸到治疗领域。居里夫人指出，放射治疗可用于某些皮肤疾病，如表浅溃疡、葡萄酒色胎记，也可用于深部恶性肿瘤，尤其是肉瘤；战伤后的畸形瘢痕、粘连、关节炎、神经炎等，也曾在战争时期的理疗中心接受治疗。

镭疗则依靠镭及其放射性衍生物释放的射线。镭的 γ 射线具有类似 X 射线的性质，而且穿透力可更强；镭管能够置于病灶附近、接触组织，甚至放入体内腔道。例如，原文特别提到子宫癌可用置入腔内的镭管治疗，关节炎也可通过放射性物质的扩散形式取得疗效。

全书最后把这些技术上升为一种社会组织问题：镭研究所等中央机构应集中镭资源，精确计量，培训专门人员，积累病例并改进技术，使癌症等顽疾的治疗不再依赖偶然资源。由此，纯科学发现最终转化为减轻痛苦、保护生命的公共事业。
