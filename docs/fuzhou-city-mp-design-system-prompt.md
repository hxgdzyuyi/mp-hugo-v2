# Fuzhou City MP Design System Prompt

> 以下路径均以本文件所在的 `docs/` 目录为基准。

请参考 [`../content/fuzhou-city/design-system.png`](../content/fuzhou-city/design-system.png) 中的视觉规范，将其作为本文的 design system，并据此完成 [`../content/mp.html`](../content/mp.html) 的文章内容填充与样式实现。

## 要求

1. 使用 [$hxgdzyuyi-skills:mp-rich-html](../../../mine-work/skills/hxgdzyuyi-skills/skills/mp-rich-html/SKILL.md) 生成微信公众号兼容的 HTML，确保最终内容可直接复制到公众号编辑器中使用。
2. 以 `design-system.png` 为视觉基准，提取其中的配色、字体层级、标题样式、卡片样式、分隔元素、图片风格和整体版式节奏。
3. 文章中的配图可以直接从 `design-system.png` 中提取视觉元素，也可以根据该设计规范，使用 [$imagegen](../../../.codex/skills/.system/imagegen/SKILL.md) 的 `image2` 模型生成风格一致的图片。
4. 填充文章正文时，请保持内容结构清晰、阅读节奏自然，并根据公众号阅读场景优化段落、标题、强调信息和视觉留白。
5. 输出结果应写入 [`../content/fuzhou-city/mp.html`](../content/fuzhou-city/mp.html)，并保持 HTML 结构简洁、稳定、兼容微信公众号渲染环境。
